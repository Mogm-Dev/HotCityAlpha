# EN-US
Welcome to HotCity, a turn-based war game!

The Alpha version of this game has two types: Alpha-A and Alpha-B.

The Alpha-A version number contains 'alphaa', and the Alpha-B version number contains 'alphab'.

The Alpha-A version of this game is an incompletely developed version. The author considered it too problematic, so it was reset and the Alpha-B version was created.

Since the Alpha-A version is very early (the first version appeared in March 2024 as 'a0.0.1a-weekly'), the author deemed it unnecessary to run both versions in parallel. Therefore, in this repository, Alpha-A only has 'aa0.1.0c-weekly' (the last weekly update version of Alpha-A).

## Schedule
The first unstable test version of Alpha-B (ab0.0.1a-weekly) is expected to be released in March 2026.

The first stable test version of Alpha-B (ab0.1.0a-monthly) is expected to be released in July 2026.

The first official release of Alpha-B (ab1.0.0a-normal) is expected to be released in September 2026.
