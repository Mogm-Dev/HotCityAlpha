# HotCityAlpha
Alpha Edition of HotCity Game.

# ZH-CN
本游戏有Alpha-A、Alpha-B两种版本，Alpha-A版本的版本号带“alphaa”，Alpha-B版本的版本号带“alphab”

本游戏的Alpha-A版本为不完全开发的版本，因作者认为其问题过多，所以对其进行了重置，制作了Alpha-B版本。

介于Alpha-A版本十分早期（2024年3月-2025年11月），作者认为没有必要作双版本同进时期，所以Alpha-A在本仓库内只有“aa0.1.0c-weekly”（Alpha-A版本的最后一个周度更新版本版本）

## 时间表（预计）

Alpha-B版本的第一个稳定测试版（ab0.1.0a-mounthly）预计于2026年7月发布
